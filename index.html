FAQ

  • What is CarpetSMP2?
    CarpetSMP2 is a Minecraft Earth server combined with a Lifesteal server, offering a unique gameplay experience on a real-world map.
  • How do I join the server?
    Use the server IP: yourserverip to connect to CarpetSMP2 in Minecraft.
  • What are the server rules?
    Rules include respecting other players, not using cheats, and following the server guidelines. For a full list, visit our server rules page.
  • How can I report issues or get help?
    You can contact our support team through the Discord server or via the in-game help commands.
  • What is Lifesteal mode?
    Lifesteal mode is a gameplay mechanic where players can gain health by defeating others, adding a unique challenge to the server.
